# Object
- It is unordered, key value pair

## When to use object?
- When we don't need order
- When we need fast access/insertion and removal

## Some Big-O value
- Insertion - O(1)
- Removal - O(1)
- Searching - O(N)
- Access - O(1)
- Object.keys - O(N)
- Object.values - O(N)
- Object.entries - O(N)
- hasOwnProperty - O(1)

# Array
- It is ordered list

## When to use array?
- When we need order
- When we need fast access / insertion and removal (sort of....)

## Some Big-O value
- Searching - O(N)
- Access - O(1)
- push - O(1)
- pop - O(1)
- shift - O(N)
- unshift - O(N)
- concat - O(N)
- slice - O(N)
- splice - O(N)
- sort - O(N * log N)
- forEach/mao/filter/reduce/etc. - O(N)
